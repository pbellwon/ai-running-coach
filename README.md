# PaceMind

> AI-powered running coach that combines training science, performance analytics and large language models to help athletes train smarter.

## Vision

Most running apps show data.

PaceMind explains the data.

The goal is not to replace a human coach, but to provide evidence-based recommendations built on:

- training history
- planned workouts
- recovery metrics
- subjective feedback
- sports science
- AI reasoning

## Goals

- Analyze training execution
- Detect long-term trends
- Identify performance limiters
- Explain every recommendation
- Adapt future training blocks
- Predict race readiness

## Data Sources

- Garmin Connect
- Intervals.icu
- Strava
- Google Sheets
- Athlete feedback

## Core Principles

- Explain every recommendation
- Never modify training without justification
- Separate calculations from AI reasoning
- Athlete data always belongs to the athlete
